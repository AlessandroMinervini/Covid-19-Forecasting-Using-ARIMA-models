# Covid-19 Forecasting Using ARIMA models
Covid-19 short-term forecasting using ARIMA models.

## Goals
- New positives
- Total positivs
- intensive care
predictions for the next 7 days.

## Orders of ARIMA models
- New positives: 3, 1, 0
- Total positivs: 3, 2, 2
- Intensive care: 1, 2, 0
